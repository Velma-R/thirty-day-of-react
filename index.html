What is react

React is JavaScript library for building user interfaces .It is the view layer for web.

At the heart of all React applications are components.

A component is a self -contained module that renders some output

Components are composable.A component might imnclude one or more other components in its output.

To write a React Appswe write react compmonents that corresponds to various interface elements.

For example take a form,A form may consist of many interface elements,like input fileds,label.or buttons.Each element inside a form can be written as a react component.

Each component in a React app abides by strict data management principles.Complex,interactive yser interfaces often involve complex data and application state.

The surface area of react is limited and aimed at giving us tools to able to anticipate how application looks with a given set of circumstances.

How does it work

React operate not directly on the browser's Document Object Model(DOM)immediately,but on a virtual DOM

When we write a react component,we are not writing directly to the DOM, but we are writing a virtual component that React will turn into the DOM.

Rather than manipulating the document in a browser after changes to our data(which can be quite slow) it resolves changes on a DOM built and run entirely in memory.

What is JSX

JSX is a syntax extension for JavaScript that lets you write HTML-like markup inside a JavaScript file

JSX and React are two separate things. Theyâ€™re often used together, but you can use them independently of each other.

JSX is a syntax extension, while React is a JavaScript library.

React component is a JavaScript function that may contain some markup that React renders into the browser

React components use a syntax extension called JSX to represent that markup.

JSX looks a lot like HTML, but it is a bit stricter and can display dynamic information

The best way to understand this is to convert some HTML markup to JSX markup
